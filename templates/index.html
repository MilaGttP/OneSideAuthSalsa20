Quarter-Round — ARX in action

Each quarter-round mixes 4 words from the 4×4 state matrix using Addition, Rotation, XOR:

b ^= ROT(a + d, 7)
c ^= ROT(b + a, 9)
d ^= ROT(c + b, 13)
a ^= ROT(d + c, 18)

Applied to columns then rows — 10 times = 20 rounds total. The output block is added back to the initial state (feed-forward), making it irreversible without the key.
